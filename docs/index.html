FAQ

Does Privacy Mesh send my data anywhere?
No. The extension makes zero external network calls. All processing happens locally in your browser. Your prompts are sanitized on-device before reaching the AI provider.
Does the AI still understand my prompt after redaction?
Yes, for most use cases. Token placeholders like [EMAIL_01] preserve meaning. If you need better AI coherence, enable Synthetic Replacement Mode — this sends realistic fake values instead of tokens.
What happens to the token map?
Token maps are stored in chrome.storage.session — scoped to a single browser tab, never synced across devices, and automatically wiped when the tab closes.
Can I trust this extension?
It's fully open source under MIT. Read every line of code on GitHub. A privacy tool's credibility is proportional to its auditability.
Which browsers are supported?
Chrome and Edge (MV3). Firefox support is included via a MV2 manifest override. Install from source until store listings are live.
